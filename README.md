# 📊 E-commerce Data Analysis

Este projeto analisa dados de e-commerce para identificar padrões de compra e prever demanda.

## 🚀 Tecnologias
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL
- Power BI / Tableau
- Machine Learning (Scikit-Learn)

## 📂 Estrutura do Projeto
- `notebooks/` → Jupyter Notebooks com análises exploratórias.
- `scripts/` → Scripts de ETL e modelagem preditiva.
- `data/` → Arquivos de dados (não incluídos no GitHub por serem grandes).

## 📊 Objetivos
- Analisar tendências de vendas.
- Criar dashboards interativos.
- Desenvolver um modelo de previsão de demanda.
